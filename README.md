## Description

Single Student Profile for NIS students. Get access to your grades, overview about school's, parallel's, class's or specific student's performance compared with any other study year. Works with SMS(school management system of NIS schools)

## Project setup

```bash
$ npm install
```

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
